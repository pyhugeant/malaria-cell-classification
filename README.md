# Malaria Cell Classification (NIH Dataset)

This project implements a deep learning pipeline for binary classification of malaria-infected red blood cells using convolutional neural networks.

---

## 1. Task Description

The goal is to classify individual red blood cell images into:

- **Parasitized** (infected)
- **Uninfected**

This task has a clear clinical relevance, as automated detection of malaria can support large-scale screening and reduce diagnostic workload.

---

## 2. Dataset

- Source: NIH Malaria Cell Images Dataset  
- Total samples: ~27,500 cell images  
- Format: RGB cropped cell images  
- Classes:
  - Parasitized
  - Uninfected
